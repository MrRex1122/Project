@workspace За каждым работником должен быть закреплен его массив задач.
Каждая задача должна содержать время и % выполнения.

Пример:
Иванов Иван,1,90
Иванов Иван,2,83
Петров Петр,1,60
Дмитриев Кирилл,4,100
Дмитриев Кирилл,6,80

У нас три критерия
Первый - количество задач у работника
Второй - среднее время выполнения задачи (сумма всех времен у всех задач разделить на количество задач)
Третий - % выполнения. Тоже средний

Значит по первому критерию, Иванов сделал две задачи
Петров сделал одну задачу
Дмитриев сделал две задачи

По второму критерию —
Время Иванова 1+2 часа = 3 часа. Было 2 задачи, значит среднее время 3/2 = 1.5 часа
Время Петрова 1 час. Задача была 1, значит среднее время 1/1 = 1 час
и тд

По третьему критерию — 
Проценты выполнения у Иванова 90 и 83. Задач было 2, значит средний процент (90+83) / 2 = 86.5
и тд
